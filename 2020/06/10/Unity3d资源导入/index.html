Unity3D资源导入

2020/06/10
Tags: unity 资源 模型 图像 uv 导入选项

在一般的游戏开发过程中，资源是开发人员接触最为频繁的东西。一个好的游戏不单单是看起来好看，同时也能够带给玩家沉浸感，好的美术，动画，音乐都会带给玩家更好的视听享受。所以资源是一个很重要的概念。当然，在好看的同时也要确保性能能够跟得上，不会出现卡顿或者是打断玩家的一些操作。所以有必要了解一些资源的基础概念和一些基础的操作，好能够更好的使用和优化我们手头上的资源。

游戏中的资源一般分为以下几种大的分类：

  1. 美术资源

    3D模型

   2D图像

   3D动画

   2D动画

  2. 音频资源

  3. 视频资源

3D模型资源

导入选项

在模型导入之后，单点模型，在Inspector面板中会有Import Setting显示

Model：模型设置

Rig：装备设置

Animation：动画导入设置

Materials：材质导入设置

首先看Model，这里分为三个部分，Scene是在场景中显示的导入设置，Meshes是关于网格的设置，Geometry是几何设置也就是关于一些贴图的设置。

Scene
Scale Factor 设置这个模型在游戏世界中的缩放比例
Convert Units 模型的单位转换，在3D文件中的比例转换unity世界中的单位
Import BlendShapes 导入文件中的混合变形器（BlendShapes是一种动画制作方式，原理就是在两个网格之间做插值运算，从上一个形状变形到另一个形状，适用于面部表情）
Import Visibility 导入文件中的可见性设置
Import Cameras 导入文件中的摄像机
Import Lights 导入文件中的灯光
Preserve Hierarchy 是否保留文件层次
Sort Hierarchy By Name 按字母表顺序在Hierarchy面板中排序
Meshes
Mesh Compression 网格压缩选项，有四个选项可以选（低中高无），用来优化网格的大小，但可能会导致网格出现错误
Read/Write Enabled 是否开始读写模式
Optimize Mesh 优化网格，确定三角形顶点的顺序，最优的选择给到GPU
Generate Colliders 生成碰撞体，这个对于静态物体可以使用
Geometry 这里主要是关于网格，UV以及贴图方面的导入选项
Keep Quads 保持网格的四边形结构，不转换为三角形
Weld Vertices 焊接顶点，相同位置的顶点将会被合并
Index Format Define the size of the Mesh index buffer.定义网格buffer的大小
Legacy Blend Shape Normals 当有Blend Shape的时候从平滑组中计算发现
Blend Shape Normals Blend Shape的法线生成方式，计算/导入/没有
Normals 法线的生成方式，计算/导入/没有
Normals Mode 定义发现在Unity中的计算方式
Smoothness Source 设置确定平滑的方式
Smoothing Angle 平滑角度，计算法线的时候用
Tangents 切线选项
Swap UVs 置换UV，当UV对象不正确的时候可以勾选此项
Generate Lightmap UVs 为光照贴图生成UV

uv

在图形学中uv指的是纹理映射；
纹理是什么？就是光栅图像，像素成为纹素，也就是纹理元素的意思；
纹理的位置通常称为纹理坐标或者是uv坐标；
纹理可能需要覆盖三维表面，从而定义了一个二维的纹理空间到三维表面的映射；
纹理空间就是纹理对应于（0，0）与（1，1）之间的矩形区域；
uv坐标是一张图像的坐标，作用是用来定位像素，查找纹理的像素；

光照贴图

光照贴图主要是对一些静态的物体的光照，是预先进行光照计算并且储存，不需要实时计算光照，适用于光照稳定的物体的贴图。说白了就是将光照的信息预先计算好了之后储存到一张贴图中，而后物体贴上光照贴图就不需要实时的灯光来照物体。

法线&切线

  1. 法线贴图？就是能够让光滑的平面有凹凸感的贴图。物体的凹凸感是通过光影来表现的，凹凸感就是是因为光照角度的不同，所以法线贴图是一种记录了需要进行光影变化的贴图，各个点的凹凸情况的贴图；
  2. 法线？法线是始终垂直于平面的线；
  3. 在物理学中，光的折射计算是通过法线来计算，那么法线贴图的作用实际上就是用来记录这些法线信息的贴图；其实实际上不是一张图，因为是借助了图像中的RGB来代表法向量中的XYZ，即为X=R，Y=G，Z=B；通过每个点的法向量信息来生成一张贴图，在进行光照计算的时候用这张图的信息来获取每个点的法向量；
  4. 法线贴图有两种模式来储存法线的信息，一种是基于模型空间，一种是基于切线空间的；
  5. 切线？就是垂直于法线的线；
  6. 基于模型空间就很好理解，就是基于模型的坐标系来定义这些点的法线；那么基于切线空间，就是在模型上某一个点，基于这个点建立坐标系，切线为X轴，法线为Y轴，副法线为Z轴；
